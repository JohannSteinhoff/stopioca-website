# Stopioca — Website (Demo)

A single-page marketing site for **Stopioca**, a boba tea shop in San Marcos, TX,
designed to match the brand from their store flyers (orange/green/yellow, rounded
display type, playful boba imagery).

**🔗 Live site:** https://johannsteinhoff.github.io/stopioca-website/

## View it
Open the live link above, or just open `index.html` in any browser — no build step,
no dependencies. Fonts load from Google Fonts; the map and images need an internet connection.

## Files
- `index.html` — page markup
- `styles.css` — all styling (brand colors defined as CSS variables at the top)
- `script.js` — mobile nav toggle + footer year
- `images/` — photos pulled from the Google Maps listing
  - `_source_urls.txt` — original image source URLs

## Before going live (notes for the sale)
- **Menu items & prices are sample/placeholder** — confirm the real menu with the owner.
- **Photos are customer-uploaded** from Google Maps. For a paid site, swap in
  original owner-provided photos (and the official logo) where possible.
- Hours, address, and phone are from the current Google listing — verify before launch.
- Add real social links (Instagram `@stopioca`, Facebook) and an online-ordering link if they use one.

## Easy deploy options
- Drag the folder onto **Netlify Drop** (netlify.com/drop) or **Cloudflare Pages**
- Or push to GitHub and enable **GitHub Pages**
- Point a domain like `stopioca.com` at it (currently unregistered)
